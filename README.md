# Binance Futures Trading Interface

A desktop application for trading Binance Futures contracts with USDT pairs.

## Features

* View all available perpetual USDT trading pairs
* Real-time Symbol List with Prices – Display all Binance Futures symbols (USDT-paired) and their latest prices.
* Trading Tab – Place long/short market orders, set TP/SL, and show selected symbol details.
* Positions Tab – Show current open positions, including PnL.
* Market Tab – General market data such as top gainers/losers, funding rates, etc.
* Basic Logging \& Config System
* Real-time price updates for selected symbols
* Place market and limit orders
* Set take profit and stop loss levels
* View current positions with PNL
* Adjust leverage

## Requirements

* Python 3.8+
* Binance API key with Futures trading enabled
